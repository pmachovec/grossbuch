\newsection{Subroutines}
\index{Subroutine}
\index{Subprogram}
\label{subroutine}
A~\mbit{subroutine}, or~also a~\mbit{subprogram}[,] is~a~repeatedly callable section of~code, which~can~take input parameters and~provide one value as~a~result.
In~other words, it's a~common name for~functions and~methods.
